# Iconos de Aplicaciones

Coloca aquí el icono de tu aplicación con las siguientes especificaciones:

## Especificaciones del Icono

- **Archivo**: `aio-icon.png`
- **Tamaño**: 512x512 píxeles (recomendado)
- **Formato**: PNG con transparencia (si aplica)
- **Tamaño de archivo**: Recomendado < 500KB

## Cómo Agregar el Icono

1. Crea o diseña tu icono de 512x512 px
2. Guárdalo como `aio-icon.png`
3. Colócalo en esta carpeta
4. El icono se mostrará automáticamente en AltStore al agregar el source

## Nota

Si tu repositorio está en GitHub, la URL del icono será:
```
https://raw.githubusercontent.com/TU-USUARIO/altstore-source/main/assets/icons/aio-icon.png
```

Esta URL ya está configurada en `apps.json`.
