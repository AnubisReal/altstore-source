# Capturas de Pantalla

Coloca aquí las capturas de pantalla de tu aplicación.

## Especificaciones

### Para iPhone
- **Tamaño recomendado**: 1170x2532 px (iPhone 12 Pro Max y superiores)
- **Formato**: PNG o JPG
- **Cantidad**: 2-5 capturas recomendadas

### Para iPad (opcional)
- **Tamaño recomendado**: 2048x2732 px (iPad Pro 12.9")
- **Formato**: PNG o JPG

## Archivos Necesarios

Nombra tus capturas de pantalla como:
- `screenshot1.png`
- `screenshot2.png`
- `screenshot3.png` (opcional)
- `screenshot4.png` (opcional)
- `screenshot5.png` (opcional)

## Consejos

1. Muestra las características principales de tu app
2. Usa capturas nítidas y bien iluminadas
3. Evita información personal o sensible
4. Optimiza el tamaño de archivo (recomendado < 1MB cada una)

## URLs en GitHub

Las URLs de las capturas serán:
```
https://raw.githubusercontent.com/TU-USUARIO/altstore-source/main/assets/screenshots/screenshot1.png
https://raw.githubusercontent.com/TU-USUARIO/altstore-source/main/assets/screenshots/screenshot2.png
```

Estas URLs ya están configuradas en `apps.json`.
